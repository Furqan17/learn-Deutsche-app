# Learn Deutsche app
A simple and responsive Andriod app to help you learn basic German :de: words.

## Includes:
* Start Screen
* Number Page 
* Colours Page

## Build 

Minimum app SDK is 21 (app intended for 5.0 Lollipop or above)  
This app was written in Java.   
Build Version 1.0
